# Task 2 (CO2) Automation Report

This repository now includes automation for **build**, **test**, **deployment**, and **CI/CD**.

## 1) Build Automation

**Tool:** Maven Wrapper (`./mvnw`)

### Implemented functionality
- Compiles source code and resolves dependencies via Maven lifecycle.
- Packages runnable Spring Boot JAR.
- Enforces reproducible build behavior with fixed `project.build.outputTimestamp`.
- Enforces Java and Maven minimum versions using Maven Enforcer plugin.

### Deliverables
- Build configuration: `pom.xml`
- Build script: `scripts/build.sh`
- Packaged artifact: `target/spring-boot-thymeleaf-example-0.0.1-SNAPSHOT.jar`

### Run
```bash
./scripts/build.sh
```

## 2) Test Automation

**Tools:** JUnit 5 (through `spring-boot-starter-test`) + JaCoCo

### Implemented functionality
- Executes unit tests with Maven Surefire.
- Generates test execution reports under `target/surefire-reports`.
- Generates test coverage report with JaCoCo under `target/site/jacoco`.

### Deliverables
- Test configuration and plugins: `pom.xml`
- Test script: `scripts/test.sh`
- Reports:
  - `target/surefire-reports`
  - `target/site/jacoco/index.html`

### Run
```bash
./scripts/test.sh
```

## 3) Deployment Automation

**Tools:** Docker + Docker Compose

### Implemented functionality
- Provides a Docker multi-stage build for a compact runtime image.
- Provides a staging profile (`application-staging.properties`).
- Automates staging deployment via shell script and `docker-compose.yml`.

### Deliverables
- Container build script: `Dockerfile`
- Staging deployment descriptor: `docker-compose.yml`
- Staging runtime config: `src/main/resources/application-staging.properties`
- Deployment automation script: `scripts/deploy.sh`

### Run
```bash
./scripts/deploy.sh
# or
docker compose up --build -d
```

## 4) Continuous Integration Setup

**Tool:** GitHub Actions

### Implemented functionality
- Triggers pipeline on push and pull request.
- Automates build + tests + coverage generation (`clean verify`).
- Publishes test and coverage artifacts.
- On `main` branch, builds Docker image and runs a smoke test container.

### Deliverables
- CI config file: `.github/workflows/ci.yml`
- Setup documentation: this report and README automation section.


## 5) Report Artifacts Explained

- **Surefire reports** (`target/surefire-reports`): XML/TXT test result files generated by Maven Surefire; used to inspect which test failed and why.
- **JaCoCo report** (`target/site/jacoco/index.html`): HTML coverage dashboard showing coverage percentages and uncovered lines.

## 6) CI Tooling Choice

- Implemented pipeline: **GitHub Actions** (`.github/workflows/ci.yml`).
- Jenkins equivalent is also provided: `Jenkinsfile`.
- You can switch to Jenkins by creating a Pipeline job pointing to this repository and selecting `Jenkinsfile` as pipeline script from SCM.
